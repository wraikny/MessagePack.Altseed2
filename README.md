# MessagePack.Altseed2

MessagePack.Altseed2 is a
[MessagePack-CSharp](https://github.com/neuecc/MessagePack-CSharp)
extension library for
[Altseed2 for .NET](https://github.com/altseed/altseed2-csharp).

## Supported Types

- Vector2F
- Vector3F
- Vector4F
- Vector2I
- Vector3I
- Vector4I
- RectF
- RectI
- Color
- Matrix33F
- Matrix44F
- Matrix33I
- Matrix44I
